使用前请知悉

  • 本程序通过 SSH 直连目标服务器执行安装：若目标服务器只能经堡垒机跳转访问、或与本机网络不可达（不同网段未打通、防火墙拦截 22 端口等），程序无法安装。
  • 安装会在目标机写入：/data/apps 下组件目录、systemd 服务、（Docker）/usr/local/bin 二进制。"中断并清理"会删除这些痕迹，请勿在生产已有同名服务的机器上随意执行。
  • 连接凭据（密码/密钥）仅在本机使用，不会上传任何第三方。
  • 目标系统支持：麒麟 / openEuler / CentOS 7+ / Ubuntu 等主流 Linux（x86_64 / arm64 自动适配；CentOS 7 老内核自动改装 Docker 20.10 兼容版）。
  • 容器组件（EMQX/RocketMQ/TDengine/Kafka/fx）需目标机先装 Docker，且本机镜像目录（默认 F:\阿克苏镜像准备）已放好对应架构镜像 tar。
  • Redis/Nginx 源码编译需目标机有 gcc/make（无网络请挂系统 ISO 配本地源，详见《无网络部署依赖清单》）。
